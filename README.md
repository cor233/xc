# 275听书网下载器

带GUI界面的275听书网音频小说下载器，模拟真实浏览器，绕过防盗链。

## 功能特点

- 🔍 搜索书籍 - 按书名、作者、演播搜索
- 📊 热门推荐 - 查看首页热门书籍
- 📥 批量下载 - 自动下载所有章节
- 🖥️ GUI界面 - 图形化操作，简单易用
- 🚀 并发下载 - 多线程加速
- 🛡️ 模拟浏览器 - 绕过防盗链限制

## 下载

从 [Releases](https://github.com/你的用户名/275-downloader/releases) 页面下载最新的 `275听书网下载器.exe`

## 使用方法

直接运行程序，在图形界面中操作：

1. **搜索下载** - 输入关键词搜索，双击结果下载
2. **热门推荐** - 查看热门书籍，双击下载
3. **设置** - 调整并发数、延迟、输出目录
